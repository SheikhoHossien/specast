# jToast
A modern & easy-going jQuery Plugin to create Toasts.

## Demo
[Click](https://l2g.cc/projects/jToast/)

## Customization options
* Text
* Duration
* Background
* Color
* Border radius

## Usage & default values
```javascript
showToast(text, {
  duration: 3000,
  background: '#232323',
  color: '#fff',
  borderRadius: '0px'
});
```

## An example
```javascript
showToast('Thanks for signing up!', {
  duration: 5000,
  background: '#20b2aa',
  color: '#f1f1f1',
  borderRadius: '15px'
});
